Ciências de dados e suas etapas 

Os processos de extração e geração de conhecimento são executadas partir das seguintes etapas normalmente: preparação dos dados, exploração dos dados, escolha do modelo, ajuste do modelo e, finalmente, avaliação e uso do modelo.

Diferença de dados estruturados com nao estruturados: A maior diferencia e que os estruturados sao organizados em formato bem definido e consistente, enquanto os nao estruturados nao segue uma estrutura especifica. 

Os dados estruturados se encaixam em dados numéricos, dados alfanuméricos ou dados categóricos, e sao usado normalmente BD relacionais e planilhas. 

Dados nao estruturados podem ser textos livres, fotos, videos etc, e normalmente e usado em NoSQL ou sistema de arquivos nao estruturados.

Em resumo a maior diferencia e a organização que pode afetar como os dados sao armazenados, gerenciados e analisados.

Dados categóricos são dados que representam uma categoria ou classificação, e podem ser divididos em duas subcategorias: nominais e ordinais.

Métodos descritivos sao uma das principais técnicas utilizadas na analise exploratória de dados em ciencia de dados.  

Os métodos preditivos são técnicas de análise de dados usadas para fazer previsões com base em padrões identificados em dados históricos. Usam algoritmo de aprendizagem de maquina

Análise preditiva em SQL é a criação de modelos de regressão linear usando funções como LINEST e FORECAST no Microsoft SQL Server para prever valores futuros com base em dados históricos em um banco de dados SQL.

Métodos supervisionados usam dados rotulados para treinar modelos de aprendizado de máquina para prever ou classificar novos dados, enquanto métodos não supervisionados identificam padrões em conjuntos de dados não rotulados. Exemplo de implementação em SQL: Clustering não supervisionado com a função K-Means para agrupar clientes com base em padrões de compras de um banco de dados SQL.

Classificação SVN 


O SVM é um algoritmo de aprendizado de máquina que classifica dados em diferentes categorias, encontrando um hiperplano que separa as amostras em duas classes distintas, maximizando a margem entre elas. Ele é comumente usado em tarefas de classificação binária e multi-classe, como na classificação de e-mails em spam ou não spam. O SVM é treinado com um conjunto de dados rotulados e usa características para classificar novos dados com alta precisão. É um algoritmo muito popular e amplamente utilizado em áreas como reconhecimento de padrões, análise de dados e processamento de linguagem natural.
